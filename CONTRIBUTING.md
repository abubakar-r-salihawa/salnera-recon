# Contributing to SALNERA Recon

Thank you for your interest in contributing! We welcome pull requests that improve the code, documentation, tests or examples. Please follow these guidelines:

1. Create an issue to discuss the proposed change before opening a pull request.
2. Fork the repository and create a new branch for your changes.
3. Write clear commit messages and reference the related issue.
4. Ensure your code follows Python style conventions (PEP8) and includes tests if appropriate.
5. Run existing tests and ensure they pass.
6. Submit a pull request describing your changes and link to the related issue.
